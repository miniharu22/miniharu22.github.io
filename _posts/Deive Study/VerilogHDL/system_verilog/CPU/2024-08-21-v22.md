- **RV321**    
  - Base Integer Instruction Set    
  - 곱셈, 나눗셈 연산이 없는 정수형 명령어 세트    

&nbsp;

<div align="left">
  <img src="/assets/images/verilog/74.png" width="70%" height="70%" alt=""/>
  <p><em></em></p>
</div>

&nbsp;

- **Instruction 구조**   
  - RV321의 Register File : 32개의 레지스터 구성   
  - 명령어 구조는 32bit 고정
    - 이는 RISC 아키텍쳐의 특징    
  - R ~ J-type이 존재    

&nbsp;

## 1. CPU Basic Module   

&nbsp;

<div align="left">
  <img src="/assets/images/verilog/13.jpg" width="70%" height="70%" alt=""/>
  <p><em></em></p>
</div>

&nbsp;

- **CPU 기본 모듈**    
  - **ALU**   
  - **Register File**    
    - 32개 레지스터   
  - **PC Register**   
    - Program Counter 레지스터   
    - Instruction이 실행할 위치(Instruction Memory)를 가지고 있는 레지스터   
  - **Instruction Memory**  
  - **Data Memory**   
    - 읽기 전용 Memory(ROM, Read Only Memory)

## 2. RV321 R-Type Instruction    

&nbsp;

<div align="left">
  <img src="/assets/images/verilog/75.png" width="70%" height="70%" alt=""/>
  <p><em></em></p>
</div>

&nbsp;

- **About Instruction**    
  - `OP Code` : Type을 나타내는 코드   
  - `Funct7`, `Funct3` : 해당 Type에서 어떤 연산을 수행하는지 설정(ALU 동작)   
  - `rd` : 연산할 값을 저장할 주소   
  - `rs` : 연산할 소스의 주소
