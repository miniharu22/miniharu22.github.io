Convolution 연산에서 Adder Stage의 기본적인 덧셈기(Adder, HalfAdder, FullAdder) 설계   

## 0. Half Adder   

```verilog
module HA(A, B, S, C); 

input A, B;
output S, C;

assign S = A ^ B;
assign C = A & B;

endmodule
```

## 1. Full Adder   

```verilog
module FA(A, B, Cin, S, Cout);

input A, B, Cin;
output S, Cout;

wire w1, w2, w3;

assign w1 = A ^ B;
assign w2 = w1 & Cin;
assign w3 = A & B;

assign S = w1 ^ Cin;
assign Cout = w2 | w3;

endmodule
```
