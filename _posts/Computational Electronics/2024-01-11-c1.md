행렬(Matrix)의 고유값(Eigenvalue)을 추출해본다.  

## 0. Generate Matrix

```python
# Import Numpy module
import numpy as np

# Generate Matrix
m0 = np.array([[2,-1], [-1,2]])

m0
```

**출력 결과** 
array([[ 2, -1],
       [-1,  2]])
{: .notice} 

&nbsp;

# Generate Matrix

```python
# Extract Eigenvalue & Eigenvector  
D, V = np.linalg.eig(m0)

# Check Eigenvalue & Eigenvector
D, V
```

**출력 결과**
(array([3., 1.]),
 array([[ 0.70710678,  0.70710678],
        [-0.70710678,  0.70710678]]))
{: .notice}
